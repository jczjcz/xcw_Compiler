### 1、常量变量声明

- 变量声明，左边有数组，右边无定义 例如int a\[2\]\[3\];  👌

- 变量声明，左边有数组，右边是初始化 例如 int a[2] = {}  int a[2] = {1,2} 👌

- 常量声明，左边有数组，右边是初始化 例如 const int a[2] = {}  const int a[2] = {1,2}  👌

- 变量声明，右边有数组 例如 int b = a\[1\]    👌
